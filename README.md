# 🎙️ EchoAI – Transcrição de Áudio com Whisper

![Python](https://img.shields.io/badge/Python-3.10+-blue)
![Streamlit](https://img.shields.io/badge/Streamlit-App-red)
![OpenAI](https://img.shields.io/badge/OpenAI-Whisper-black)
![Status](https://img.shields.io/badge/Status-Active-green)

Aplicação web para transcrição de áudio utilizando a API Whisper da OpenAI.

Permite transformar fala em texto a partir de diferentes fontes:

- 🎤 Microfone em tempo real  
- 🎧 Arquivos de áudio (.mp3)  
- 🎬 Vídeos (.mp4)

---

##  Stack

Python · Streamlit · OpenAI Whisper API · PyDub · MoviePy · WebRTC

---

##  Como executar

```bash
pip install -r requirements.txt
streamlit run app.py
